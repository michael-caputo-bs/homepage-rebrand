FAQ

Questions, Answered

What is the Betstamp Sports Betting API?
A single API that delivers normalized, real-time betting data — odds, player props, alternate lines, game props, futures, and schedules — from 200+ sportsbooks, exchanges, and prediction markets. One integration, one schema, every source.
Which sportsbooks and markets do you cover?
200+ sources: 191 sportsbooks, 31 DFS/pick'em operators, 9 exchanges, and 11 prediction markets. We cover 2,500+ markets across the NFL, NBA, MLB, NHL, college football and basketball, soccer (EPL, La Liga, UCL and more), UFC, tennis, and others — including main lines, player props, game props, alternates, and futures.
How do I get the data — REST or streaming?
Both. Pull on demand via REST, or subscribe to a real-time Server-Sent Events stream for push updates. Median refresh is 400ms with a 99.99% uptime SLA. You can also access everything visually through the PRO Odds Screen.
How fast can we integrate?
Most teams are pulling live data the same day. Grab an API key, hit a single endpoint, and you get a clean, consistent JSON response with stable IDs across every book — plus SDKs and full docs to move fast.
